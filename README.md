# P0_wireframe
Benvingut a GuitarShop Som una botiga especialitzada en guitarres, on trobaràs les guitarres més luxoses, sempre amb la millor qualitat i servei. Si ets un guitarrista principiant o un professional experimentat, tenim alguna cosa per a tu. Navega pel nostre catàleg, descobreix les nostres ofertes i aprèn més sobre cada instrument.

Inici:
Presentació de la botiga. Amb un botó desplegable que mostra el menú. També hi ha una secció de destacats amb una breu descripció de cada guitarra. Enllaça amb el menú i amb la pàgina a detall de cada guitarra al clicar a sobre d'una imatge.

<img src = imatges/1.jpg width="200" height="250">

Menú: 
Desplegable amb totes les seccions de la pàgina web per poder navegar facilment. Aquest menú enllaça amb el catàleg, la cistella, el blog i la pàgina d'inici de sessió/registre.

<img src = imatges/2.jpg width="200" height="250">

Blog:  
Secció per publicar articles sobre guitarres i notícies del sector.

<img src = imatges/3.jpg width="200" height="250">

Catàleg:
Pàgina on es troben totes les guitarres a disposició del client amb un filtre per poder trobar la guitarra adequada. Cada guitarra conte amb una fotografia, una breu descripció, el seu preu i les opcions extres del producte. Al clicar a una guitarra t'envia a la pàgina en detall de la guitarra. I al clicar al filtre accedeixes a un desplegable per ajustar la teva búsqueda.

<img src = imatges/4.jpg width="200" height="250">

Filtre:
Aquesta secció conte amb un filtre de preu, marca i tipus de guitarra. Al clicar al botó d'aplicar t'envia de nou al catàleg.

<img src = imatges/5.jpg width="200" height="250">

Detall:
Al clicar a una fotografia del catàleg t'envia a aquesta secció. Aqui és troba informació detallada de cada guitarra: fotografies de la guitarra, especificacions tècniques, ressenyes del clients i un botó per afegir el producte a la cistella.
